What do you need to know about the project?

Here are the main parts of the project

Hardware

The hardware, currently an ESP32, considered the heart of the project as it has the function of sending and receiving data.

Display

The Display is also a key part of the project, currently using the common LCD display together to project the legend directly to the user.

Capitation microphone

The microphone is responsible for capturing all the ambient audio and transferring it to the hardware, currently using a model to the Inmp441.

Body finish

To connect all the pieces, a group developed a glass to serve as a finishing touch to the project currently made of hardened cardboard.
